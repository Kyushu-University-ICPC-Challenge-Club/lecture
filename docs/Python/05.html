# 5. 繰り返し

プログラミングでは、決まった回数やある条件を満たしている間は同じ処理を繰り返し行う場合があります．

Python では、繰り返しには while for を使用します．

# [ 1 ] while 文

while 文を使うと、ある条件が成り立つ間、処理を繰り返し実行するというプログラムが書けます．

while 文は次のように使用します．

while 条件式:
    処理
1
2

上のコードでは、条件式 が成り立つとき 処理 を繰り返し実行します．

条件式の後にコロン (:) が必要なことに注意してください．

if 文と同様に、インデントにより、繰り返し実行する処理を指定します．

コード 1 では、x が正の数の間、処理を繰り返し実行します．

# コード 1

x = 5
while x > 0:
    print(x)
    x = x - 1

# continue 文

continue 文を使用することで、ループ処理を途中でスキップして、再びループの最初に戻ることができます．

continue 文は多くの場合、if 文と組み合わせて次のコードように使用します．

条件式 2 が真となるとき、continue 文を実行して 処理 2 をスキップして、再びループの最初に戻ります．

while 条件式 1:
    処理 1
    if 条件式 2:
        continue
    処理 2

# [ 2 ] for 文

for 文を使うと、決められた回数だけ、処理を繰り返し実行するというプログラムが書けます．

for 文は次のように使用します．

for 変数 in range(開始値, 終了値):
    処理
1
2

上のコードでは、終了値 - 開始値 回数分だけ 処理 を繰り返し実行します．

range(開始値, 終了値) の後にコロン (:) が必要なことに注意してください．

変数 の値は、ループする毎に 開始値 から 終了値 - 1 まで 1 ずつ増加します．

コード 5 では、5 回 print(i) を実行します．

# コード 5

for i in range(0, 5):
    print(i)

問題 5-4 の解答
a, b, c = map(int, input().split())
ans = 0
for i in range(a, b + 1):
    # 「c を i で割った余りが 0」↔︎「i は c の約数」
    if c % i == 0:
        ans += 1
print(ans)

# 多重ループを抜け出す

break 文を使用することで 1 つのループを抜け出すことができますが、複数のループを抜け出すにはフラグを使用します．

例えばコード 6 の九九のプログラムで、i = 1, j = 5 のときに 2 つのループを抜け出したときは、コード 7 のように i = 1, j = 5 に辿り着いたかを表すフラグを用意して抜け出します．

# コード 7

flag = False  # 「ループを抜け出す」用のフラグ

for i in range(1, 10):
    for j in range(1, 10):
        print(f'{i} × {j} = {i * j}')

        # i = 1, j = 5 に辿り着いた
        if i == 1 and j == 5:
            flag = True

        # 2 番目のループを抜ける
        if flag == True:
            break

    # 1 番目のループを抜ける
    if flag == True:
        break

# 問題 5-6 : フレームの描画 (opens new window)


問題 5-6 の解答
while True:
    H, W = map(int, input().split())

    # H = 0 かつ W = 0 のとき終了する
    if H == 0 and W == 0:
        break

    for i in range(H):
        if i == 0 or i == H - 1:
            # 1 行目と H 行目は # のみ
            for j in range(W):
                print('#', end='')
        else:
            # 2 ~ H - 1 行目は 1 列目と W 列目が # でそれ以外は . となる
            for j in range(W):
                if j == 0 or j == W - 1:
                    print('#', end='')
                else:
                    print('.', end='')
        print()

    # データセットごとに改行する
    print()

# 問題 5-7 : チェスボードの描画 (opens new window)


問題 5-7 のヒント
  • 行と列の偶奇に注目してみましょう．

while True:
    H, W = map(int, input().split())

    # H = 0 かつ W = 0 のとき終了する
    if H == 0 and W == 0:
        break

    for i in range(H):
        for j in range(W):
            if (i + j) % 2 == 0:
                print('#', end='')
            else:
                print('.', end='')
        print()

# 問題 5-8 : 組み合わせの数 (opens new window)


問題 5-8 の解答
while True:
    n, x = map(int, input().split())
    if n == 0 and x == 0:
        break
    ans = 0
    for i in range(1, n + 1):
        for j in range(i + 1, n + 1):
            for k in range(j + 1, n + 1):
                if i + j + k == x:
                    ans += 1
    print(ans)
