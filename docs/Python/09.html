# 9. 集合

集合は重複しない要素を持つリストのようなものです．

コード 1 のように複数のデータを {} で囲みます．

# コード 1

S = {1, 2, 3, 1, 2, 3, 4}
print(S)  # {1, 2, 3, 4}
1
2

# [ 1 ] 集合の要素数

リストや文字列と同様に len(集合) で集合の要素数を求めます．

# コード 2

S = {1, 2, 3, 4}
print(len(S))  # 4
1
2

# 要素の追加

集合.add(要素) で集合に要素を追加します．

# コード 3

S =  {1, 2, 3, 4}
S.add(5)
print(S)  # {1, 2, 3, 4, 5}
1
2
3

# [ 2 ] 要素の検索

in 演算子で要素の検索ができます．

要素 in 集合 が True の場合、集合に指定する要素が含まれて、False の場合、含まれていません．

# コード 4

S = {1, 2, 3, 4}
print(2 in S)    # True
print(100 in S)  # False
1
2
3

# [ 3 ] 要素の削除

集合.discard(要素) で集合から指定する要素を削除します．

また、集合.clear() で集合から全ての要素を削除します．

# コード 5

S = {1, 2, 3, 4}

S.discard(3)
print(S)  # {1, 2, 4}

S.clear()
print(S)  # {}

# [ 4 ] リスト・文字列を集合に変換する

set 関数を使用して、文字列やリストを集合に変換することができます．

# コード 6

L = [3, 1, 4, 1, 5]
print(set(L))  # {1, 3, 4, 5}

S = "banana"
print(set(S))  # {'n', 'b', 'a'}

S = set()
print(S)  # {}

# リストのユニーク化

list(set(リスト)) で指定したリストの重複する要素を削除 (ユニーク化) します．

# コード 7

L = [3, 1, 4, 1, 5]
L = list(set(L))
print(L)  # [1, 3, 4, 5]

# 問題 9-1 : Set: Search (opens new window)


問題 9-1 の解答
S = set()
q = int(input())
for i in range(q):
    c, x = map(int, input().split())
    if c == 0:
        S.add(x)
        print(len(S))
    else:
        print(int(x in S))
