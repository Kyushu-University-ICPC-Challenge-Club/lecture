# 10. 辞書

辞書はキーと値のペアを管理するオブジェクトです．

集合と同じように {} を使って表します．

{キー1: 値1, キー2: 値2, ...} のように、キーと値の対応を決めます．

辞書[キー] で指定したキーに対応する値を求めます．

ただし、辞書に存在しないキーを指定するとエラーになります．

# コード 1

D = {'Alice': 90, 'Bob': 85, 'Carol': 70}
print(D['Alice'])  # 90
print(D['Bob'])    # 85
print(D['Carol'])  # 70
print(D['Dave'])   # エラーが発生する

# 辞書の要素数

リストや文字列と同様に len(辞書) で辞書の要素数を求めます．

# コード 2

D = {'Alice': 90, 'Bob': 85, 'Carol': 70}
print(len(D))  # 3

# 要素が空の辞書
D2 = {}
print(len(D2))  # 0

# 要素の追加・変更

辞書[キー] = 値 で指定したキーに対応する値を決めます．

# コード 3

D = {'Alice': 90, 'Bob': 85, 'Carol': 70}

# 要素の追加
D['Dave'] = 100
print(D)

# 要素の変更

# 辞書と for 文

# 各要素のキーに対してループ処理

for 変数 in 辞書.keys() で 変数 に 辞書 のキーを代入しながらループを回します．

# コード 5

D = {'Alice': 90, 'Bob': 85, 'Carol': 70}
for i in D.keys():
    print(i)

# リストから辞書を作成する

dict(zip(リスト1, リスト2)) でリスト 1 の要素をキー、リスト 2 の要素を値とする辞書を作成します．

# コード 7

L1 = ['Alice', 'Bob', 'Carol']
L2 = [90, 85, 70]
D = dict(zip(L1, L2))
print(D)

# 問題 10-1 : Map: Search (opens new window)


問題 10-1 の解答
D = {}
q = int(input())
for i in range(q):
    L = input().split()
    if L[0] == '0':
        key, x = L[1], L[2]
        D[key] = x
    else:
        key = L[1]
        print(D[key])

# 問題 10-2 : Map: Search (opens new window)


問題 10-2 の解答
D = {}
q = int(input())
for i in range(q):
    L = input().split()
    if L[0] == '0':
        key, x = L[1], L[2]
        D[key] = x
    elif L[0] == '1':
        key = L[1]
        if key in D.keys():
            print(D[key])
        else:
            print(0)
    else:
        key = L[1]
        if key in D.keys():
            del D[key]
