# 3. 入力

# [ 1 ] input 関数

input 関数を使用することで、ユーザからの入力を受け取ることができます．

変数 = input() で、キーボードから入力した文字列が変数に代入されます．

# コード 1

weight = input()
print('私の体重は {weight} kgです．")
1
2

# 入力 1

62.5
1

# 出力 1

私の体重は 62.5 kgです．
1

input 関数について、2 つ注意点があります．

# 注意点 1

1 つ目の注意点は、input 関数は 1 行丸ごと入力として受け取ることです．

例えばコード 1 が入力 2 を受け取ったとき、weight には '62.5 170' が入るため、
出力 2 のように 2 つの値が一緒に出力されます．

値を切り離して扱いたい場合は、複数の値を受け取る場合 で解説します．

# 入力 2

62.5 170
1

# 出力 2

私の体重は 62.5 170 kgです．
1

# 注意点 2

2 つ目の注意点は、input 関数は入力されたデータを 文字列型 のデータとして返すことです．

下の例をみてみましょう．

変数 A には入力 123 が入りますが、A の中身は文字列 '123' となります．

そのため、コード 3 の 2 行目では 文字列 + 数値 の計算を行うことになりエラーが発生します．

# コード 3

A = input()
B = A + 100  # A は文字列、100 は数値なのでエラーが発生する
1
2

# 入力 3

123
1

# 出力 3

TypeError: can only concatenate str (not "int") to str
1

このようなエラーを回避するためには、次に紹介する 型変換 を行います．

# [ 2 ] 型変換

データの型を変換することを 型変換 といいます．

型を整数型に変換するには、int 関数を使用します．

int(データ) でデータの型を整数型に変換したものを返します．

# コード 4



 



A = '123'

A = int(A)   # A を文字列型から整数型に変換

B = A + 100  # A は整数、100 は数値なのでエラーが発生しない
1
2
3
4
5

# 出力 4

223
1

型変換を使うことで先程の input 関数の注意点 2 の問題を解決できます．

# コード 5

A = int(input())  # 入力と型変換をまとめて書くこともできます
B = A + 100
1
2

# 入力 5

123
1

# 出力 5

223
1

TIP : 文字列への変換

逆に、数値を文字列型に変換するには str 関数を使用します．

B = 456
print(A + B)  # 579

B = str(B)
print(A + B)  # 123456 (文字列 123 と文字列 456 の連結)

# 問題 3-2 : 来月は何月？ (opens new window)


問題 3-2 の解答
N = int(input())
print(N % 12 + 1)
1
2
  • 1 年は 12 月周期です．周期が関係する計算は余り % を使うと大体上手くいきます．
  • N % 12 を計算すると、12 月が 0 月になるので、N % 12 + 1 が答えになります．

# [ 3 ] 複数の値を受け取る場合

複数の値を受け取る場合を具体例を通して学習します．

例として、整数 A と B を受け取り、A + B を出力するプログラムを書くことにします．

Python では 入力の形 によってコードの実装を変える必要があります．

入力の形として次の 2 パターンがあります．

  • パターン 1 : 2 行でひとつずつ与えられるパターン
A
B
1
2
  • パターン 2 : 1 行に並べて与えられるパターン
A B
1

# パターン 1

コード 6 のように input 関数を 2 回連続で使用すればいいです．

# コード 6

 
 


A = int(input())    # A の値を受け取る + 型変換
B = int(input())    # B の値を受け取る + 型変換
print(A + B)
1
2
3

# パターン 2

コード 7 のように A, B = input().split() で入力を受け取った後、個別に型変換をします．

int(input().split()) だとエラーになるので注意してください．

# コード 7

 
 
 


A, B = input().split()  # A と B の入力を受け取る
A = int(A)              # A の型変換
B = int(B)              # B の型変換
print(A + B)
1
2
3
4

TIP : map 関数

入力として 1 行に 5 個の整数が並べて与えられる場合を考えます．

コード 7 の通りにすると、入力を受け取るプログラムは次のようになります．

# コード 8

a, b, c, d, e = input().split()
a = int(input())
b = int(input())
c = int(input())
d = int(input())
e = int(input())
1
2
3
4
5
6

これだと型変換の部分を 5 行書く必要があり大変です．

これは、map 関数を使うことでコード 9 のようにシンプルに書くことができます．

# コード 9

a, b, c, d, e = map(int, input.split())
1

これは、input().split() で受け取った各文字列に対して、
int 関数を適用する (整数に型変換をする) という処理を行っています．

# 練習問題

# 問題 3-3 : スワップ (opens new window)


問題 3-3 の解答
a, b = map(int, input().split())
print(b, a)
1
2

# 問題 3-4 : Apple Pie (opens new window)


問題 3-4 の解答
A, P = map(int, input().split())
print((3 * A + P) // 2)
1
2

# 問題 3-5 : お茶 (opens new window)


問題 3-5 の解答
A, B = map(int, input().split())
print((B + A - 1) // A)
1
2
  • B / A の小数点を切り上げた整数は (B + A - 1) // A と表せます．
    実際に具体的な値を入れてみて確かめてみましょう．
